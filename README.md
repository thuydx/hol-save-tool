This is a project to make to tools to edit the save files of the ***House of Legacy*** game. 
Based on [Next.js](https://nextjs.org/) and [CoreUI](https://coreui.io/).

Environment used:
 - npm: 11.6.2
 - node: v25.2.1
 - yarn: 4.12.0
 - Next.js: v16

## Getting Started

### Create a `.env.local` file
```dotenv
 SASS_PATH=./node_modules/@coreui/coreui/scss;./src/styles
```

### Install dependencies
```bash
npm install
# or
yarn
# or
pnpm install
```

### Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
